Data Mining

Data mining or data exploration is a field of statistics and computer science referring to the process that attempts to discover patterns in large volumes of data sets. It uses the methods of artificial intelligence, machine learning, statistics and database systems.




Definicion

It is defined as the process of detecting significant patterns in data. Data mining is a complement natural to the process of exploiting and understanding data. Automatic algorithms can process very large amounts of data and detect patterns and trends that would otherwise be hidden.


Objetivos principales

  • Convert data into knowledge to make decisions.
  • Extract information from a set of data and transform it into an understandable structure for later use.
